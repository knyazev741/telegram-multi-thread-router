   - HTML formatted message body
   - Tool name in `<code>` tags
   - Input summary: `json.dumps(input_data, indent=2)[:500]` + "..." if truncated

4. **PermissionManager** class:
   - `__init__`: `self._pending: dict[str, asyncio.Future] = {}`
   - `create_request() -> tuple[str, asyncio.Future]`: generate uuid4, create future via `asyncio.get_running_loop().create_future()` (NOT `get_event_loop()` -- Python 3.14 requirement per Pitfall 3), store in `_pending`, return `(request_id, future)`

   - `expire(request_id: str) -> None`: pop from `_pending` without resolving. Called on timeout.
   - `pending_count` property: return `len(self._pending)` (useful for testing)

All imports at top. Use `from __future__ import annotations` for clean type hints.

   ```python
   def __init__(self, thread_id: int, workdir: str, bot: Bot, chat_id: int,
   ```

4. **Add `_can_use_tool` method** to SessionRunner (replaces _auto_allow_tool):
   ```python
   async def _can_use_tool(self, tool_name: str, input_data: dict, context) -> PermissionResultAllow | PermissionResultDeny:
   ```
   - First check: `if tool_name in self._allowed_tools: return PermissionResultAllow(updated_input=input_data)` (PERM-07 auto-approve)
   - Save `prev_state = self.state`, set `self.state = SessionState.WAITING_PERMISSION`
   - Call `request_id, future = self._permission_manager.create_request()`
   - Send permission message: `await self._bot.send_message(chat_id=self._chat_id, message_thread_id=self.thread_id, text=format_permission_message(tool_name, input_data), parse_mode="HTML", reply_markup=build_permission_keyboard(request_id))`
   - Wrap in try/except/finally:
     - try: `action = await asyncio.wait_for(future, timeout=300.0)` (PERM-05)
     - except TimeoutError: call `self._permission_manager.expire(request_id)`, send timeout message "⏱ Permission timed out — denied" to topic, return `PermissionResultDeny(message="Timed out — user did not respond within 5 minutes")`
     - finally: `self.state = prev_state` (Pitfall 4 — always restore state)
   - Handle action results:
     - "allow": return `PermissionResultAllow(updated_input=input_data)`
     - "always": add to `self._allowed_tools`, return `PermissionResultAllow(updated_input=input_data, updated_permissions=[PermissionUpdate(type="addRules", rules=[PermissionRuleValue(tool_name=tool_name)], behavior="allow")])` (PERM-06)
     - "deny": return `PermissionResultDeny(message="Denied by user")`

5. **Update `_run` method** — change `can_use_tool=_auto_allow_tool` to `can_use_tool=self._can_use_tool` in ClaudeAgentOptions.

6. **PRESERVE** the `_dummy_pretool_hook` function and its registration in hooks dict (PERM-09). Do NOT modify the hooks line.
