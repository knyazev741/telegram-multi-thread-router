**SessionRunner class:**

Constructor `__init__(self, thread_id: int, workdir: str, bot: Bot, chat_id: int, session_id: str | None = None, model: str | None = None)`:
- Store all params as instance attributes
- `self.state = SessionState.IDLE`
- `self._client: ClaudeSDKClient | None = None`
- `self._message_queue: asyncio.Queue[str | None] = asyncio.Queue()` (None is stop sentinel)
- `self._task: asyncio.Task | None = None`

Method `async def start(self) -> None`:
- `self._task = asyncio.create_task(self._run())`

Method `async def _run(self) -> None`:
- Build system_prompt via `_build_system_prompt(self.workdir)`
- Create `ClaudeAgentOptions` with:
  - `cwd=self.workdir`
  - `model=self.model` (None = SDK default)
  - `system_prompt=system_prompt`
  - `can_use_tool=_auto_allow_tool`
  - `hooks={"PreToolUse": [HookMatcher(matcher=None, hooks=[_dummy_pretool_hook])]}`
  - `resume=self.session_id` (None for new sessions)
  - `include_partial_messages=True`
- `async with ClaudeSDKClient(options=options) as client:`
  - `self._client = client`
  - Loop: `while self.state != SessionState.STOPPED:`
    - `text = await self._message_queue.get()`
    - `if text is None: break` (stop sentinel)
    - `self.state = SessionState.RUNNING`
    - `await client.query(text)`
    - `await self._drain_response(client)`
    - `if self.state == SessionState.INTERRUPTING: self.state = SessionState.STOPPED; break`
    - `self.state = SessionState.IDLE`
    - `await update_session_state(self.thread_id, "idle")`
- Wrap the entire `async with` in try/except. On exception: log error, set state=STOPPED, send error message to topic via `self._bot.send_message(chat_id=self._chat_id, message_thread_id=self.thread_id, text=f"Session error: {e}")`.
- In finally: `self._client = None`

Method `async def _drain_response(self, client: ClaudeSDKClient) -> None`:
- `async for msg in client.receive_response():`
  - If `isinstance(msg, AssistantMessage)`:
    - For each block in `msg.content`:
      - If `isinstance(block, TextBlock) and block.text`:
        - `await self._bot.send_message(chat_id=self._chat_id, message_thread_id=self.thread_id, text=block.text)`
  - If `isinstance(msg, ResultMessage)`:
    - If `self.session_id is None and msg.session_id`:
      - `self.session_id = msg.session_id`
      - `await update_session_id(self.thread_id, msg.session_id)`
    - `logger.info("Turn complete for thread %d: cost=$%s, duration=%dms", self.thread_id, msg.total_cost_usd, msg.duration_ms)`

Method `async def enqueue(self, text: str) -> None`:
- `await self._message_queue.put(text)`

Method `async def stop(self) -> None`:
- `prev_state = self.state`
- `self.state = SessionState.INTERRUPTING`
- If `self._client and prev_state == SessionState.RUNNING`:
  - `await self._client.interrupt()`
  - (Drain happens inside `_run` loop — the current `_drain_response` will complete and the loop checks state)
- `await self._message_queue.put(None)` (sentinel to unblock queue.get())
- `await update_session_state(self.thread_id, "stopped")`
- If `self._task:` `await asyncio.wait_for(self._task, timeout=10.0)` wrapped in try/except asyncio.TimeoutError (cancel task on timeout)

Property `@property def is_alive(self) -> bool`:
- Return `self._task is not None and not self._task.done()`
