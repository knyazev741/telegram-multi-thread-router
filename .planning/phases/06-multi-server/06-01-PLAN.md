2. **src/ipc/server.py** — Create with:

   `class WorkerRegistry`:
   - `__init__`: `self._workers: dict[str, asyncio.StreamWriter] = {}`
   - `register(worker_id, writer)`: store writer
   - `unregister(worker_id)`: pop writer
   - `async send_to(worker_id, msg) -> bool`: get writer, check `writer.is_closing()`, call `send_msg(writer, msg)`, return True. Return False if not found or closing.
   - `is_connected(worker_id) -> bool`: check key exists and not closing
   - `list_workers() -> list[str]`: return list of connected worker_ids

   `async def start_ipc_server(host, port, auth_token, bot, session_manager, permission_manager, worker_registry) -> asyncio.Server`:
   - Uses `asyncio.start_server` with a `handle_connection` callback
   - `handle_connection` spawns `asyncio.create_task(_handle_worker(...))` — never blocks the accept loop
   - Returns the Server object

   `async def _handle_worker(reader, writer, auth_token, bot, session_manager, permission_manager, worker_registry)`:
   - Read first message with `recv_w2b(reader)` — must be AuthMsg
   - If not AuthMsg or token != auth_token: send `AuthFailMsg(reason="invalid token")`, close writer, return
   - Otherwise: send `AuthOkMsg(worker_id=msg.worker_id)`, register in WorkerRegistry
   - Enter message loop: `while True: msg = await recv_w2b(reader); if msg is None: break`
   - Dispatch by message type using isinstance:
     - `AssistantTextMsg`: call `split_message(msg.text)` then `bot.send_message(chat_id=settings.group_chat_id, message_thread_id=msg.topic_id, text=part, parse_mode="Markdown")` for each part. Catch exceptions and retry without parse_mode.
     - `PermissionRequestMsg`: call `permission_manager.create_request()` to get (request_id, future). Send Telegram message with `format_permission_message(msg.tool_name, msg.input_data)` and `build_permission_keyboard(request_id)`. Create an asyncio task that awaits the future (with 300s timeout), then sends `PermissionResponseMsg(request_id=msg.request_id, action=result)` back via TCP. On timeout, send action="deny".
     - `StatusUpdateMsg`: (log for now — full status wiring deferred to plan 03)
     - `SessionStartedMsg`: log worker_id + session_id
     - `SessionEndedMsg`: send error message to topic if msg.error, log otherwise
     - `McpSendMessageMsg`: `bot.send_message(chat_id=settings.group_chat_id, message_thread_id=msg.topic_id, text=msg.text)`
     - `McpReactMsg`: `bot.set_message_reaction(chat_id=settings.group_chat_id, message_id=msg.message_id, reaction=[ReactionTypeEmoji(emoji=msg.emoji)])`
     - `McpEditMessageMsg`: `bot.edit_message_text(chat_id=settings.group_chat_id, message_id=msg.message_id, text=msg.text)`
     - `McpSendFileMsg`: `bot.send_document(chat_id=settings.group_chat_id, message_thread_id=msg.topic_id, document=FSInputFile(msg.file_path), caption=msg.caption)`
   - In `finally`: unregister from WorkerRegistry, close writer
