  <action>
Create `src/sessions/mcp_tools.py` with a factory function `create_telegram_mcp_server(bot: Bot, chat_id: int, thread_id: int)` that returns an MCP server with 4 tools.

The function:
1. Calls `server = create_sdk_mcp_server("telegram")` to create an MCP server instance.
2. Registers 4 tools using the `@server.tool()` decorator pattern (each tool is a closure over bot, chat_id, thread_id):

**Tool 1: reply(text: str) -> str**
- Description: "Send a text message to the user in the current Telegram thread"
- Calls `await bot.send_message(chat_id=chat_id, message_thread_id=thread_id, text=text)`
- Returns "Message sent" on success, error string on failure

**Tool 2: send_file(path: str) -> str**
- Description: "Send a file to the user in the current Telegram thread. Path must be absolute or relative to working directory."
- Check file exists with `Path(path).exists()` — return error if not
- Check file size with `Path(path).stat().st_size` — if > 50MB (50 * 1024 * 1024), return "File exceeds 50MB Telegram limit"
- Use `from aiogram.types import FSInputFile` and `await bot.send_document(chat_id=chat_id, message_thread_id=thread_id, document=FSInputFile(path))`
- Returns "File sent: {filename}" on success, error string on failure

**Tool 3: react(emoji: str, message_id: int) -> str**
- Description: "Add an emoji reaction to a specific message in the thread"
- Calls `await bot.set_message_reaction(chat_id=chat_id, message_id=message_id, reaction=[ReactionTypeEmoji(emoji=emoji)])`
- Returns "Reaction added" on success, error string on failure

**Tool 4: edit_message(text: str, message_id: int) -> str**
- Description: "Edit a previously sent message in the thread"
- Calls `await bot.edit_message_text(chat_id=chat_id, message_id=message_id, text=text)`
- Returns "Message edited" on success, error string on failure

Each tool wraps its body in try/except, catching Exception and returning `f"Error: {e}"`.

3. Returns the `server` object.

Imports: `logging`, `from pathlib import Path`, `from aiogram import Bot`, `from aiogram.types import FSInputFile, ReactionTypeEmoji`, `from claude_agent_sdk import create_sdk_mcp_server`.
