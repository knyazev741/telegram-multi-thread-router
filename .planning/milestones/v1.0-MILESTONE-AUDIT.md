### Bug Found and Fixed

**BUG-001** (MSRV-05, fixed in ae73a92): `_await_permission` in `src/ipc/server.py` captured `msg.request_id` by closure reference instead of by parameter value. Under concurrent permission requests, the wrong request_id would be sent back to the worker. Fixed by passing `msg.request_id` as explicit parameter.
